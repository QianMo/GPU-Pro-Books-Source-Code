Description:
Calculates the tangent space for all supplied vertices. Writes tangent and bitangent vectors to the output vertices, copies all other elements from input vertices. Will split vertices if necessary - i.e. if two triangles sharing a vertex want to assign it different tangent-space matrices. The decision whether to split uses fSplitDifference - of the DP3 of two desired tangents or two desired bitangents is higher than this, the vertex will be split.
